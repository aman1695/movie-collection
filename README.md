# movie-collection
## Step to Run App
1. Install node
2. Install mongodb
3. run git clone https://github.com/aman1695/movie-collection.git
4. cd movie-collection
5. run npm install
6. run mongod
7. run npm start
8. go to http://localhost:3000/
